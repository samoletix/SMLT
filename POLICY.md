Политика использования (GD Leaderboard — SMLT Edition)
1. Источник данных

Все статистические данные (очки, позиции в глобальном топе, записи прохождений) подтягиваются в реальном времени через открытое API Global Demonlist (demonlist.org). Сайт является лишь визуальной оболочкой для отображения конкретной группы игроков.
2. Критерии попадания в список

На данный момент проект является закрытым комьюнити-листом. Чтобы попасть в таблицу, необходимо:

    Быть участником стаффа SMLT.

    Иметь подтвержденные очки/рекорды в официальном GDL.

    По поводу добавления писать администратору: @.samoletik (Discord) или @samoltik (Telegram).

3. Обновление данных

Данные обновляются при каждой загрузке страницы. Если вы только что прошли уровень и его приняли на основном Demonlist, может потребоваться небольшое время для кэширования API, прежде чем он появится здесь.
4. Ответственность и модерация

    Администрация SMLT не несет ответственности за ошибки на стороне основного API demonlist.org.

    Любые попытки манипуляции данными (если они будут обнаружены в обход официального листа) приведут к исключению из локального лидербоарда.
