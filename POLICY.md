Политика использования и конфиденциальности SMLT
1. Общие положения

    Сайт SMLT (SMLT Community Edition) является некоммерческим проектом сообщества Discord.

    Сайт создан исключительно для развлекательных целей: ведения таблицы лидеров (Демонлиста) Geometry Dash и демонстрации совместных проектов участников.

    Используя данный сайт, вы автоматически соглашаетесь с условиями настоящей политики.

2. Какие данные мы собираем и зачем

    Данные профилей Geometry Dash: Сайт автоматически запрашивает публичную информацию (ники, очки, места в топе, рекорды и страны) через сторонний API (api.demonlist.org) на основе списка игроков, который формируется Хостом сервера.

    Локальное хранилище (LocalStorage / SessionStorage): Сайт использует память вашего браузера исключительно для сохранения выбранной вами темы оформления (светлая/тёмная) и сессии авторизации Хоста. Мы не используем рекламные куки (cookies) и не отслеживаем ваши действия.

3. Права участников и удаление данных

    Все данные о рекордах и игроках берутся из открытых публичных источников (Demonlist API).

    Если вы хотите попасть на этот лист, то напишите админу:

        Discord: @.samoletik

        Telegram: @samoltik

    Ваш профиль или проект будут удалены с сайта в кратчайшие сроки по вашему первому требованию.

4. Ограничение ответственности

    Администрация сайта не несёт ответственности за возможные сбои в работе стороннего API (demonlist.org) или неточность отображения глобальных очков/рангов.

    Администрация не несёт ответственности за содержание видеороликов, ссылки на которые привязаны к проектам, если они были изменены на сторонних платформах (например, на YouTube).

5. Изменения политики

    Настоящая политика может обновляться по мере добавления новых функций на сайт. Актуальная версия всегда доступна на этой странице.
